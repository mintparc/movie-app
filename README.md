## Git Commit Rule
`use Gitmoji`

 * 🎉 `:tada:` begin a project

 * ✨ `:sparkles:` when introduce new features
 * 🐛 `:bug:` when fixing a bug 
 * 🔧 `:wrench:` add or update configuration files.
 * 🔥 `:fire:` when removing code or files
 * ♻️ `:recycle:` when refactor code
